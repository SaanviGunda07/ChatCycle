# ChatCycle Wireframe

**ChatCycle Wireframe** is a Flutter-based UI prototype for a social/chat app. It demonstrates the layout, navigation, and interactions without requiring backend functionality. All data and assets are stored locally.


## **Project Overview**

* Built using **Flutter** for cross-platform UI and state management.
* Provides a **bottom navigation bar** with four main pages: Explore, Posting, Notifications, and Profile.
* Uses **local assets** (images and icons) for fast load times and placeholder content.
* Designed as a **wireframe** â€” focuses on layout, interactions, and visual structure.


## **Pages and Features**

1. **Explore / Trending Page**

   * Displays trending posts and questions in scrollable cards.
   * Includes like, comment, and share icons for UI interaction.

2. **Posting Page**

   * Form with dropdowns for chat and subject selection.
   * Choice chips for keywords and a comment box for detailed posts.
   * Floating action button to create new posts.

3. **Notifications Page**

   * Lists recent notifications with avatars, text, and timestamps.

4. **Profile Page**

   * Banner image and user avatar at the top.
   * Displays username, description, follow button, and collections section with thumbnails.

---

## **Assets**

* All images and icons are stored locally in the `assets/` folder.
* Includes profile avatars, post thumbnails, collection cards, icons, and notification icons.

---

## **Navigation and UI**

* Bottom navigation bar with interactive icons.
* Floating action button to quickly access the Posting page.
* Scrollable lists and tapable cards to mimic real app interactions.

## **Getting Started**

1. Clone the repository:

```bash
git clone <repo-url>
```

2. Install dependencies:

```bash
flutter pub get
```

3. Run the app:

```bash
flutter run
```
